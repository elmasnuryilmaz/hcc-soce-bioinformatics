# Code Availability

Analysis code and reproducibility documentation for the manuscript **"Prognostic Significance of TRPC6 and STIM1 in Hepatocellular Carcinoma: An Integrative Bioinformatic Analysis"** are available at:

<https://github.com/elmasnuryilmaz/hcc-soce-bioinformatics>

The repository contains numbered analysis modules, public-data source notes, software requirements, and a reproducibility runbook. Each numbered folder contains the analysis script and the associated output tables or figures used to support the manuscript.

## Suggested Manuscript Statement

Analysis code and reproducibility documentation are available in the project repository: <https://github.com/elmasnuryilmaz/hcc-soce-bioinformatics>. The repository includes the numbered analysis scripts, `requirements.txt`, a reproducibility runbook, and README documentation describing public data acquisition and analysis execution.

## Reproducibility Scope

- Public data sources are listed in [data_sources_and_methods.md](data_sources_and_methods.md).
- Execution order and expected outputs are listed in [REPRODUCIBILITY_RUNBOOK.md](REPRODUCIBILITY_RUNBOOK.md).
- The manuscript-facing analysis modules are mapped in [README.md](README.md).
- No protected patient-level data or private credentials are included.
